# HoTools

Sistema de gestão de hotel desenvolvido com Next.js e Supabase.

## Tecnologias

- Next.js
- React
- TypeScript
- Tailwind CSS
- Supabase

## Instalação

```bash
npm install
```

## Executar

```bash
npm run dev
```

## Variáveis de ambiente

Crie um `.env.local`:

```
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
```

## Build

```bash
npm run build
npm run start
```
